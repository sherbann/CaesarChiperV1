Ceaser Cipher program

Encoding:

  1. Select the values for your two keys, Key1 and Key2 (keep these values in mind for the decoding process later);
  2. Select the Encode button;
  3. Input the text message that you want to encode. (using letters);
  4. The encoded text message will be visible in the output field bellow;
  5. On the output field, you can now find the encoded text, ready to use.

Decoding:

  1. Select the values for your two keys that you used to encode your message;
  2. Select the Decode button;
  3. Type or paste in the encoded text message;
  4. On the output field, you can now find the original text message.
